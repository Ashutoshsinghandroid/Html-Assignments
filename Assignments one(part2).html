Self- Closing tags:

A self closing tags in HTML are the type of HTML tags that need not to be closed manually by its closing tag, which means there is no saperate closing tag for it .
Self closing tags are also alternatively known as void tags, empty tags, singletons tags, these tags do not have contents and also can not have any child. example :- img tag and br tags

Meta tags

The tag defines metadata about an HTML document. Metadata is data (information) about data.Metadata is used by browsers (how to display content or reload page), search engines (keywords), and other web services.
