Short note on tags , elements and attributes:-

Tags :-
In html tags are used to define the structure of web page , there are certain types of tags, but mostly tags comprises with opening and closing brackets, some the tags are self closing tags , for eg. br tag is self closing tag while p tag body tag having closing brackets.
Elements :-
Html elements comprises of opening tag , content and closing tag, collection of all these three together called as Html elements eg:- heading tag it comprises opening brackets, the name of heading and then closing brackets.
Attributes :-
Each tag can have certain set of funtioning with the help of which we can define the behavior of tag, it is alawys applied with start tag for eg: style and title are some example of attributet

Any 3 tags and give brief explanation:-

A tags:- The HTML anchor tag defines a hyperlink that links one page to another page. It can create hyperlink to other web page as well as files, location, or any URL. The "href" attribute is the most important attribute of the HTML a tag. and which links to destination page or URL.
img tag:- HTML img tag is used to display image on the web page. HTML img tag is an empty tag that contains attributes only, closing tags are not used in HTML image element.
ol tags:- HTML Ordered List or Numbered List displays elements in numbered format. The HTML ol tag is used for ordered list. We can use ordered list to represent items either in numerical order format or alphabetical order format, or any format where an order is emphasized. There can be different types of numbered list:
Numeric Number (1, 2, 3) Capital Roman Number (I II III),Small Romal Number (i ii iii),Capital Alphabet (A B C) Small Alphabet (a b c)
